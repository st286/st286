## Linux kernel compile

安装编译工具以及其它一下依赖的软件包，在Ubuntu 18.04环境下执行如下命令
```
sudo apt-get install git fakeroot build-essential ncurses-dev xz-utils libssl-dev bc flex libelf-dev bison
```
