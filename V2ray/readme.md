
## 安装 Vr2ay

```shell
  # installation:
  bash <(curl -L -s https://install.direct/go.sh)
  
  # Upgrade:
	 bash <(curl -L -s https://install.direct/go.sh) -c

  # Help:
	bash <(curl -L -s https://install.direct/go.sh) -h

```

## 启动v2ray

```shell
	service v2ray start 
	
	systemctl start v2ray
	
	restart  enabled

```
